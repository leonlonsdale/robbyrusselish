
Each palette customizes the following prompt elements:

-   error
-   success
-   directory
-   branch
-   symbol
-   status
-   python
-   node
-   rust
-   go

## Usage

1. Put starship.toml in `~/.config`
2. If you put it elsewhere, remember to set your `STARSHIP_CONFIG` var:
    ```sh
    export STARSHIP_CONFIG="$HOME/.config/starship/starship.toml"
    ```
3. Set your preferred palette in `starship.toml`:
    ```toml
    palette = "ayumirage"
    ```
